Triangulation formula applied

Confidence % = base_weight + recency_bonus + stakeholder_bonus. Base weight: 1 source = 35, 2 sources = 60, 3 sources = 80, 4 sources = 90, 5 sources = 95. Recency bonus = +5 if signal within 14 days. Stakeholder bonus = +5 if VP+ mentioned the signal. Cap at 95%.
